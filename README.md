# 🎬 IMDb Movie Recommendation System

## 📌 Project Overview
This project recommends movies based on storyline similarity using NLP and Machine Learning.

## 🚀 Features
- Recommend movies using movie name
- Recommend movies using storyline
- Built using Streamlit

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- NLP (TF-IDF, Cosine Similarity)

## 📂 Project Structure
```
IMDb_Project/
│
├── app/
├── data/
├── model/
├── scripts/
```

## ▶️ Run Project
```
cd app
streamlit run app.py
```

## 📊 Output
Top 5 similar movies based on input
